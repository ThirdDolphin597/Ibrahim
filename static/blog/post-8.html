Intro

This week I made significant progress on the LED lightbox project. After discovering a critical issue with the hall effect sensor, I had to remake the PCB and reassemble everything. While there's still work to be done, the project is coming together nicely.

The Hall Effect Sensor Problem

When I first tested everything, the LEDs worked perfectly and the toggle switch functioned as expected, but the hall effect sensor wasn't responding at all. I went through the circuit trying to figure out what went wrong. After some investigation, I realized the issue—the hall effect sensor pin was floating. The pull-up resistor wasn't even attached, so the input was reading random noise instead of clean signals. It needed to be connected to the switch. This was a significant enough problem that I had to remake the entire PCB to fix it properly.

Milling the New PCB

I started by milling the new PCB with the corrected circuit design. This time, I made sure the pull-up resistor was properly connected in the layout. I measured everything carefully and milled it on the CNC.

Milling the new PCB with the fixed design.

Final board assembly ready for soldering.

Assembly and Soldering

I started by supergluing all the plastic connectors to the PCB to make everything solid and stable. Once the glue set, I soldered all the connections. Then I added the two 10k resistors, the button, and the capacitor to complete the circuit.

Soldering components to the PCB.

Completed PCB

After all the soldering was complete, the PCB was fully assembled with all the components securely attached. The plastic connectors were solidly bonded to the board, and all the electronic components—including the resistors, button, and capacitor—were properly soldered in place.

Completed PCB with all components soldered and attached.

Testing the Edge Logo

I laser cut and etched a couple of different versions of the Edge logo to see which would work best. I made some black and white versions as well as a color one to test different design options. After comparing them, the color version came out best (bottom right).

Testing different laser-etched versions of the Edge logo.

The New Tray

With the new PCB design, I needed a new tray to match the updated screw holes. I printed a fresh tray that will allow me to mount everything properly. Soon I'll mount all the components together and install the magnets.

The new tray with matching screw holes for the updated PCB.

Next Steps

The PCB is complete and the logo design is finalized, but there's still work ahead. I need to screw the PCB into the tray to secure it properly, and then mount the switch, LED strip, and hall effect sensor. Once those are installed, I'll be able to do final testing and complete the assembly.
